CFDT Graphics Page Browser — Improved Tray

WINDOWS
1. Double-click Launch_CFDT_Graphics.bat
2. Your browser should open automatically.
3. Keep the launcher window open while using the app.

MAC
1. Double-click Launch_CFDT_Graphics.command
2. If macOS blocks it, right-click the file and choose Open.
3. Keep Terminal open while using the app.

WORKFLOW
1. Search or filter the plan pages.
2. Select a page.
3. Choose a graphic type.
4. Click Add to graphics tray.
5. In the tray, rename, regenerate, delete, or click the thumbnail to return to the source page.

The tray is saved locally in your browser, so it remains after closing and reopening the launcher.
